# 1. Employee Salary Calculation from CSV

### 1.1 Calculate Total Salary of the Employees by Reading a CSV File

- **Task**:  
  Read a CSV file containing employee data and calculate the total salary of all employees.

- **Steps**:
    1. Read the CSV file.
    2. Extract the salary column.
    3. Sum up all the salaries to get the total salary of all employees.

---

### 1.2 Calculate Total Salary Based on Their Designations from a CSV File

- **Task**:  
  Read a CSV file containing employee data and calculate the total salary for employees based on their designations.

- **Steps**:
    1. Read the CSV file.
    2. Group employees by their designation (e.g., H.R, Software Developer, Marketing).
    3. For each designation, sum up the salaries of employees belonging to that group.
    4. Example Output:
        - Total Salary for H.R
        - Total Salary for Software Developers
        - Total Salary for Marketing
